## РАЗДЕЛ 2: ЛУЧШИЕ ПРАКТИКИ ДИЗАЙНА

### Цветовые схемы лидеров

- **Jasper AI**: тёмно-фиолетовый (#4A154B) + оранжевый акцент, dark mode как основной
- **Copy.ai**: фиолетовый (#6366F1, Indigo) + белый, минималистичный B2B
- **Writesonic**: синий (#0EA5E9) + фиолетовый градиент, яркий tech-startup
- **SMMplanner**: синий (#2563EB) + белый, понятный для RU-пользователя

**Тренд 2024-2025:**
- Тёмные фоны (dark mode) + градиентные акценты — у 7 из 10 топовых AI-платформ
- Фиолетово-синяя гамма = ассоциация с AI/технологиями
- "Glassmorphism" эффекты (стеклянные карточки с blur)
- Анимированные фоны (частицы, нейронные сети, плавные градиенты)

### Типографика

| Платформа | Шрифт | Размер H1 |
|---|---|---|
| Jasper AI | Inter Bold | 56-72px |
| Copy.ai | Cal Sans / Inter | 60px |
| Writesonic | Plus Jakarta Sans | 56px |
| Notion | Inter | 48px |

**Рекомендованный стек для 2025:** Inter + Onest (отличная кириллица) или Unbounded для H1

### UI-компоненты которые повторяются

1. **"Live demo"** — интерактивная демонстрация прямо на главной
2. **Logo wall** — логотипы клиентов (серые, монохромные)
3. **Анимированный текст** в hero (смена слов: "статьи / посты / рассылки")
4. **Dashboard-скриншот** показывающий интерфейс продукта
5. **Gradient blur blob** как декоративный элемент фона
6. **Счётчики** (X миллионов пользователей, Y тысяч контента)
7. **Сравнение "до/после"**

## РАЗДЕЛ 4: ТОП-5 ФИШЕК ДЛЯ "КОД КОНТЕНТА"

### Фишка 1: AI в действии прямо на сайте
Поле ввода "Введите тему для поста" → моментальная генерация примера без регистрации → "Хотите полную версию? Зарегистрируйтесь"

### Фишка 2: Счётчик сэкономленного времени
"Нейросотрудники Код контента уже создали 127 840 постов — это 2 560 часов сэкономленного времени"

### Фишка 3: Сценарный онбординг по ролям
Кнопки-роли → "Я владелец магазина" / "Я SMM-специалист" / "Я маркетолог" → разные потоки/примеры

### Фишка 4: Кейсы с конкретными цифрами
Карточки ДО/ПОСЛЕ: "Агентство X тратило 40 часов/неделю на контент — теперь 8 часов"

### Фишка 5: Живое демо нейросотрудника
Встроенный чат-виджет с демо-ботом прямо на лендинге — конкуренты этого не делают

## РАЗДЕЛ 6: ДИЗАЙН-СИСТЕМА

### Рекомендованная цветовая схема (тёмная)

```
Фон hero:        #0A0A0F
Акцент 1:        #7C3AED (фиолетовый — AI)
Акцент 2:        #F59E0B (янтарный — премиум)
Текст основной:  #F9FAFB
Текст вторичный: #9CA3AF
Карточки:        rgba(255,255,255,0.05) + border rgba(255,255,255,0.1)
CTA кнопка:      градиент #7C3AED → #A855F7
```

### Типографика

```css
/* Заголовки */
font-family: 'Unbounded', sans-serif; /* для H1 */
font-family: 'Onest', sans-serif;     /* для остального */
H1: 56-72px, font-weight: 800, letter-spacing: -0.02em

/* Тело */
font-family: 'Onest', sans-serif;
font-size: 16-18px, line-height: 1.6
```

### Анимации

| Элемент | Эффект |
|---|---|
| Hero текст | Typewriter-смена слов ("посты / статьи / боты") |
| Счётчики | Анимированный инкремент при скролле |
| Карточки | Hover: подъём + свечение |
| Фон hero | Градиентный mesh / CSS blob-анимация |
| Секции | Fade-in при скролле (AOS / Framer Motion) |

## ИТОГ: КЛЮЧЕВЫЕ ВЫВОДЫ

**Что объединяет все топовые платформы:**
1. Тёмный hero + фиолетово-синяя гамма — стандарт AI-ниши
2. Конкретные цифры везде — не "быстро", а "за 30 секунд"
3. Демо без регистрации — снижает барьер входа радикально
4. 3 плана тарифов — не больше, не меньше
5. Логотипы клиентов — даже 5-10 реальных логотипов работают
6. Одна суперчёткая ценность — одна боль → одно решение

**Уникальные возможности "Код контента" vs конкурентов:**
- Русский рынок — все международные конкуренты плохо локализованы
- Концепция "нейросотрудник" сильнее чем "инструмент"
- Telegram-фокус — самая актуальная для РФ соцсеть
- Комплексность — контент + боты + автопостинг в одном месте
